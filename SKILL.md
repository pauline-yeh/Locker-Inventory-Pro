# Locker Search (儲物櫃搜尋)

此技能協助使用者在電子儲物櫃中快速定位物料。

## 工作流程

1. **讀取最新資料**：
   - 檢查桌面上的 `電子儲物櫃.xlsx` 是否存在。
   - 使用 `scripts/read_data.ps1` 將 Excel 資料轉換為最新的 `locker_data.json`。
   
2. **生成搜尋介面**：
   - 讀取 `locker_data.json`。
   - 根據資料內容生成一個包含搜尋功能的 HTML Artifact。
   - 介面必須符合「Web Application Development」的高質感標準：
     - 使用現代配色（例如：深色主題或優雅的藍色調）。
     - 具備即時搜尋過濾功能。
     - 以卡片或表格形式清晰展示品名、規格、存放位置等。

## 注意事項
- 如果 `locker_data.json` 已存在且使用者沒有要求重新整理資料，可以直接使用現有 JSON。
- 搜尋時應同時過濾「品名」與「規格」。
