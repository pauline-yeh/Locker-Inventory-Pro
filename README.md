# Locker Inventory Pro - 儲物櫃物料查詢系統

這是一個專業、直覺且具備高度互動性的電子儲物櫃物料查詢系統，旨在提升物料庫存管理與尋找的效率。

## 🌟 核心特色

### 1. 精確的位置對應系統
- **1-to-1 精準對應**：基於最新的 Excel 數據結構，確保每一個物料卡片都能精確連結到圖表中的特定小方塊。
- **雙向互動導航**：
  - 點擊物料卡片：側邊欄圖表會自動高亮並定位到對應位置。
  - 點擊圖表方塊：主畫面會立即過濾並顯示該方塊記憶體放的所有物料。

### 2. 直覺的視覺設計 (Premium UI)
- **色系連動**：儲物櫃的每一層都有獨特的配色，並同步反映在物料卡片的邊框與底色上。
- **高亮視覺提示**：選中的位置會以「紅色粗框 + 呼吸燈脈衝動畫」顯示，極大化視覺辨識度。
- **層級路徑顯示**：側邊欄會動態顯示當前選擇的完整路徑（例如：置物櫃(上) > 第一層 > 方塊A）。

### 3. 即時庫存管理功能
- **領取與歸還**：每張物料卡片均配置「領取 (-)」與「歸還 (+)」功能鍵。
- **即時數值連動**：在庫存數量變動時，介面會立即反應最新的數量，無需重新載入網頁。
- **馬卡龍美感按鈕**：控制區域採用了薄荷綠馬卡龍色系與圓潤的圖示化設計，提升使用愉悅感。

### 4. 進階搜尋與過濾
- **多維度搜尋**：支援透過品名、規格、料號或存放位置關鍵字進行模糊搜尋。
- **一鍵重設**：整合的「清除篩選」按鈕可立即清空所有搜尋條件與位置過濾。

## 🛠️ 技術架構

- **前端**：Vanilla HTML5, CSS3 (Modern Flexbox/Grid Layout), JavaScript (ES6+).
- **視覺效果**：動態呼吸燈動畫 (CSS Keyframes), 高質感陰影與毛玻璃質感背景。
- **數據同步**：支援透過 PowerShell 腳本自動從 Excel (`.xlsx`) 提取數據並生成單一 HTML 介面。
- **環境適應**：側邊欄圖表強制固定於 100vh 且不產生捲軸，優化大螢幕的使用體驗。

## 🚀 如何使用

1. **開啟程式**：直接在瀏覽器開啟 `儲物櫃搜尋介面.html`。
2. **快速搜尋**：在上方搜尋欄輸入關鍵字。
3. **定位物料**：點擊感興趣的物料卡片，左側會顯示其在櫃子裡的確切位置。
4. **管理數量**：使用卡片上的「領取」與「歸還」按鈕來更新手邊的庫存數量。
